【快速命令速查】

启动应用
  npm run start:dev

数据库迁移
  npx prisma migrate dev --name migration_name
  npx prisma migrate deploy
  npx prisma migrate status

查看数据库
  npx prisma studio

代码格式化
  npm run format

代码检查
  npm run lint

运行测试
  npm run test

生产构建
  npm run build
  npm run start:prod
